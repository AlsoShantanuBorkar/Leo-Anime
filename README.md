## Run Locally

*Make sure Flutter SDK is installed*


- Clone the project

```bash
  git clone https://link-to-project
```

- Go to the project directory

```bash
  cd my-project
```

- Check for errors before running app

```bash
  flutter doctor
```

*Make sure a device is present to build the app on*

- Build App 

```bash
  flutter run
```
